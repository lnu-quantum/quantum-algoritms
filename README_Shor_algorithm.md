# Алгоритм Шора 

### Опис

Цей проєкт містить реалізацію квантового алгоритму Шора для факторизації натурального числа N із використанням бібліотеки **Qiskit** та симулятора **AerSimulator**.

Алгоритм складається з трьох основних етапів:

1. **Класична підготовка**  
   Перевірка властивостей числа N (парність, простота, perfect power), вибір випадкового значення a та попередній факторинг через `gcd(a, N)`.

2. **Квантова частина — Quantum Phase Estimation (QPE)**  
   Побудова унітарного оператора для модульного множення  
   `U(y) = a * y mod N`  
   та застосування QPE для оцінювання фази, з якої визначається порядок r числа a за модулем N.

3. **Класична постобробка**  
   Використання знайденого порядку r для отримання нетривіальних дільників числа N за формулами:  
   - `gcd(a^(r/2) - 1, N)`  
   - `gcd(a^(r/2) + 1, N)`

## Файли

**`ShorAlgorithm.ipynb`** — основний ноутбук, який містить:
- реалізацію всіх частин алгоритму Шора,
- приклад факторизації числа N,
- побудову QPE-схеми,
- аналіз виміряної фази та отриманих дільників.
