# Алгоритм Гровера на Qiskit (2 кубіти)

Цей проєкт демонструє реалізацію квантового **алгоритму Гровера** для пошуку одного "позначеного" елемента в невпорядкованій базі з 4 елементів (2 кубіти) за допомогою бібліотеки **Qiskit**.

Алгоритм Гровера дає квадратичне прискорення порівняно з класичним лінійним пошуком.

## Файли

- `grover.py` — основний скрипт:
  - будує оракул для цільового стану (`'00'`, `'01'`, `'10'` або `'11'`);
  - будує оператор дифузії;
  - конструює повну схему алгоритму Гровера;
  - запускає її на симуляторі `qasm_simulator` (qiskit-aer);
  - виводить схему та статистику вимірювань.

## Вимоги

- Python 3.10+ (рекомендовано 3.12)
- Пакети:
  - `qiskit`
  - `qiskit-aer`

Встановлення:

```bash
pip install "qiskit>=1.0.0" qiskit-aer
